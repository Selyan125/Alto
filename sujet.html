🎯 Objectif du Projet

Développer une application mobile permettant à deux utilisateurs d'établir une connexion sécurisée via un système de pairing par QR Code, puis d'échanger des informations chiffrées de bout en bout.

Backend Fourni ✓

Serveur : Spring Boot disponible sur https://alto.samyn.ovh

Votre rôle : Développer l'application mobile Flutter

🖼️ Les 4 Écrans Principaux

1️⃣ HomeScreen

  • Créer une connexion
  • Scanner un QR code
  • Liste des relations
  • Sélection active

2️⃣ InitPairingScreen

  • Génération du QR code
  • Indicateur chargement
  • Polling (attente scan)
  • Timer d'expiration

3️⃣ ScanPairingScreen

  • Vue caméra plein écran
  • Cadre de visée
  • Permissions caméra
  • Feedback visuel/sonore

4️⃣ RelationScreen

  • Historique messages
  • Saisie message
  • Types multiples
  • Auto-refresh 5 sec

🟢 Partie 1 : Pairing Initial

4 points

1.1 - Initialisation (1 pt)

Clés RSA • UUID • QR code • Stockage sécurisé

1.2 - Scan et Match (1 pt)

Caméra • QR parsing • API PUT /pairing • Récupération données

1.3 - Polling (1 pt)

Timer 2-3 sec • GET /pairing/status • Détection completion • Timeout 2 min

1.4 - Finalisation (1 pt)

API DELETE • Récupération infos • Confirmation • Persistance

🟡 Partie 2 : Échange Sécurisé

8 points

2.1 - Chiffrement (2 pts)

  • Interface saisie message

2.2 - Déchiffrement (2 pts)

2.3 - Types Multiples (4 pts)
